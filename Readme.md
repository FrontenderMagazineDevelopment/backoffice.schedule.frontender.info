# Backoffice Schedule @ fronteder.info

## First start

1. You should firstly build application:
    ```
      yarn build
    ```
2. Then start application with simple:
    ```
      yarn start
    ```

3. Open [http://localhost:3000/](http://localhost:3000/)

## Development, build and deploy

  * Run application

    ```
      yarn start
    ```
  * Development

    ```
      yarn dev
    ```

  * Build

    ```
      yarn build
    ```
